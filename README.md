Pizza Ordering App (Spring Boot MVC)

This is a web application built using Spring Boot, Thymeleaf, and the MVC architecture.
Users can create pizza orders, apply toppings, choose delivery options, and view order history.

Features:
- Order form with validation
- Price calculation with tax and discount logic
- Order summary page
- Order history page
- In-memory storage using a service layer

Built for PROG27545 Assignment 1.
